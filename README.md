# TODO
We claim that this is the simplest To Do list organizer.

  
To list tasks type `./todo`:
```
$ todo
=======================
    TODO:              
    -> Make a new bridge on bosphorus
    -> rm -rf /*
=======================
```
  
To add a task `./todo "<new task>"`:

```
$ todo "Do extreme ironing @12:03 p.m"
=======================
    TODO:              
    -> Make a new bridge on bosphorus
    -> rm -rf /*
    -> Do extreme ironing @12:03 p.m
=======================
```
